SIS-BIBLIO (Sistema de Gestión Bibliotecaria) is a complete platform for managing a library's book catalog, loan records, and user accounts. It enforces professional-grade security at every layer and is built entirely with PHP 8+, HTML5, CSS3, vanilla JavaScript, and MySQL via PDO — no external frameworks.

## User roles

SIS-BIBLIO distinguishes between two roles:

- **Administrator (Staff)** — full access to all modules: book catalog, loan management, reader profiles, and user account administration.
- **Reader (Lector)** — limited access to their own loan history and account details.

## Architecture overview

```text
SIS-BIBLIO/
├── modules/       # Dashboard pages and CRUD forms
├── config/        # Core logic: PDO connection, auth, CSRF
├── assets/        # Public CSS and images
│   └── dashboard/ # Per-page encapsulated CSS for the dashboard
└── PHPMailer/     # Transactional email engine
```

Each module in `modules/` is a self-contained PHP page. Configuration and authentication logic live in `config/`, keeping concerns cleanly separated.
