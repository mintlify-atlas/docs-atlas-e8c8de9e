## Overview

The loans module at `modules/prestamos.php` tracks the complete lifecycle of each book loan — from the moment a book leaves the library to when it is returned or cancelled.

- **Administrators** see all loans in the system.
- **Readers** see only their own loans.

The table displays: Loan code, Reader, Book title, Loan date, Return date, Status, and available actions.

---

## Loan status values

Each loan has a status badge that reflects its current state:

| Status | Meaning |
|--------|---------|
| **Pendiente** | Loan registered but not yet confirmed as active |
| **Confirmada** | Loan is active — the book has left the library |
| **Cancelada** | Loan was cancelled before or during the loan period |

## Registering a loan

Both administrators and readers can register new loans.

<Steps>
  <Step title="Open the loan form">
    Click the **Registrar Préstamo** button at the top of the loans page. This opens `modules/prestamos_form.php`.
  </Step>
  <Step title="Review the loan code">
    A unique loan code (e.g. `LIB-4823`) is auto-generated by the system using the pattern `LIB-XXXX`. The code is displayed for reference but cannot be edited.
  </Step>
  <Step title="Select the reader">
    Choose the **Lector Solicitante** from the dropdown. Administrators can select any active reader. Readers can only select themselves (their own linked reader profile).
  </Step>
  <Step title="Select the book">
    Choose the **Libro a Prestar** from the dropdown. Only books with **Disponible** status appear in this list.
  </Step>
  <Step title="Set the dates">
    Set the **F. Préstamo** (loan date, defaults to today) and **F. Límite/Devolución** (return date, defaults to 7 days from today).
  </Step>
  <Step title="Submit">
    Click **Generar Vale** to register the loan. The new loan is always created with **Pendiente** status. An administrator can update the status later by editing the loan.
  </Step>
</Steps>

<Info>
  You must have an active reader profile linked to your account before you can register a loan. If no active reader card is found for your account, you will see an error message. Contact an administrator to set up your reader profile.
</Info>

## Cancelling a loan

Any authenticated user can cancel a loan that is in **Pendiente** or **Confirmada** status.

Click the **ban icon** on the loan row. A browser confirmation dialog appears. Click **OK** to confirm. The loan status changes to **Cancelada** and remains in the record for audit purposes.

<Note>
  Cancelling a loan does **not** automatically update the book's status. If the book was marked **Ocupada**, an administrator must manually edit the book and set its status back to **Disponible**.
</Note>

---

## Editing a loan (admin only)

Click the **pencil (edit) icon** on any loan row. The loan form opens with the current values pre-filled. When editing, administrators can modify the reader, book, dates, and also the **Estado** (status) field — which only appears on the edit form, not when creating a new loan. Click **Generar Vale** to save.

---

## Deleting a loan (admin only)

Click the **red trash icon** on any loan row. A browser confirmation dialog appears. Click **OK** to permanently remove the loan record from the database. This action cannot be undone.
