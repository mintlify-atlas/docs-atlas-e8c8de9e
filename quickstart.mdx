This guide walks you through signing in, adding a book, registering a reader, and recording your first loan — the core workflow of SIS-BIBLIO.

<Note>
  Make sure you have completed [installation](/installation) and imported the database before following these steps.
</Note>

## Step 1: Sign in as administrator

Navigate to your SIS-BIBLIO installation and open the login page:

```
http://localhost/CRUD-SIS-BIBLIO-Publico/login.php
```

Use the pre-seeded administrator account:

| Field | Value |
|-------|-------|
| Email | `admin@ejemplo.com` |
| Password | `1234567` |

Click **Ingresar**. You are redirected to `modules/dashboard.php`.

<Warning>
  Change these default credentials before deploying to any non-local environment.
</Warning>

## Step 2: Review the dashboard

The dashboard shows three stat cards for administrators:

- **Total Libros Catálogo** — total books in the system
- **Lectores Registrados** — total reader profiles
- **Préstamos Activos (Global)** — active and pending loans

The left sidebar provides access to all modules. On mobile, tap the **☰** hamburger button to expand the sidebar.

## Step 3: Add a book

<Steps>
  <Step title="Open the catalog">
    Click **Catálogo** in the left sidebar. The book list at `modules/libros.php` appears.
  </Step>
  <Step title="Create a new book">
    Click the blue **Nuevo Libro** button in the top right. The form at `modules/libros_form.php` opens.
  </Step>
  <Step title="Fill in book details">
    Enter the following fields:

    | Field | Description |
    |-------|-------------|
    | Título | Book title (must be unique) |
    | Autor | Author name |
    | Precio | Replacement cost (decimal, e.g. `150.00`) |
    | Estado | Availability status — choose **Disponible** |

  </Step>
  <Step title="Save the book">
    Submit the form. The book appears in the catalog with a green **DISPONIBLE** badge.
  </Step>
</Steps>

## Step 4: Register a reader

<Steps>
  <Step title="Open the readers list">
    Click **Lectores** in the sidebar to open `modules/lectores.php`.
  </Step>
  <Step title="Add a new reader">
    Click **Nuevo Lector** and fill in:

    | Field | Description |
    |-------|-------------|
    | Nombre completo | Reader's full name |
    | Teléfono | Contact phone number |
    | Estado | Set to **Activo** |

  </Step>
  <Step title="Save the reader">
    Submit the form. The reader appears in the list and is now available to assign loans.
  </Step>
</Steps>

<Tip>
  Readers who register themselves via the public `register.php` page are automatically added to the system with the Reader role and create their own reader profile — no manual action required.
</Tip>

## Step 5: Create a loan

<Steps>
  <Step title="Open loans">
    Click **Mis Préstamos** in the sidebar to open `modules/prestamos.php`.
  </Step>
  <Step title="Register a new loan">
    Click **Registrar Préstamo** and complete the form:

    | Field | Description |
    |-------|-------------|
    | Código Generado | Auto-generated loan identifier (e.g. `LIB-4823`) — read-only, filled by the system |
    | Lector | Select the reader you just created |
    | Libro | Select the book you just added |
    | Fecha préstamo | Loan start date (defaults to today) |
    | Fecha devolución | Expected return date (defaults to 7 days from today) |

  </Step>
  <Step title="Confirm the loan">
    Submit the form. The loan appears in the list. When the book physically leaves the library, update the status to **Confirmada**.
  </Step>
</Steps>
